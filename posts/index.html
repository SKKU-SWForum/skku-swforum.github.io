Inuiyeji cluster

What is Inuiyeji cluster? ‘Inuiyeji’ cluster is a server cluster that is available for SKKU students who major in Software Engineering. Mobaxterm

How to use Mobaxterm Mobaxterm is a program that supports running remote programs such as ssh, sftp, and X11 forwarding. Mobaxterm has a free and paid version, but the free version is more than enough. Here’s how to install: Go to https://mobaxterm.mobatek.net/ to download and install the installer. Open the program. Press ‘session’ at the top left. Press the leftmost ‘ssh’ and set it up. remote host: (Example) swji.
