Mobaxterm

Mobaxterm 사용법 Mobaxterm은 ssh, sftp, X11 forwarding 등 원격 프로그램 실행, 명령어 사용 등을 지원하는 프로그램입니다. 무료 버전과 유료 버전이 있지만, 무료 버전만으로도 충분히 활용할 수 있습니다. 다음은 설치 방법입니다. https://mobaxterm.mobatek.net/ 에 접속해서 설치 프로그램을 다운로드받고 설치합니다. 프로그램을 켭니다. 좌측 상단의 세션을 누릅니다. 맨 왼쪽 ssh를 누르고 설정을 합니다. remote host: (예시) swji.skku.edu Port: (예시) 1398 OK를 누르고 좌측에 생성된 세션을 더블클릭하면 ssh가 연결됩니다.

VSCode

VSCode - Remote SSH 연동 Visual Studio Code(이하 VSCode)는 다양한 플러그인을 활용할 수 있는 강력한 텍스트 에디터입니다. Mobaxterm을 대체하여 VSCode를 인의예지 서버와 연동하여 다양한 기능을 활용할 수 있습니다. 설치 방법 VSCode가 설치되지 않은 경우, 여기를 클릭하여 VSCode를 설치합니다. VSCode를 켜고 아이콘을 클릭합니다. 새로 열린 창에서 Remote - SSH를 검색하고 나오는 플러그인을 설치합니다. 왼쪽 상단 아이콘을 클릭합니다. 위의 Remote Explorer가 SSH Target인지 확인합니다. SSH TARGETS 옆에 있는 + 버튼을 클릭합니다. 중앙 상단에 SSH Connection Command를 입력하라는 창이 뜹니다.
